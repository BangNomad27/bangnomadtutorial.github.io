Mulai

Apa itu Tag? Tag adalah sebuah penanda atau kode yang dibuat dengan kurung patah ( <...> ), dan didalamnya terdapat namaTag-nya dan terdapat juga atributnya (pake jika dibutuhkan). Nah, Tag itu sendiri ada Tag pembuka dan Tag penutup (<...>namaTag</...>).

Penggunaan NamaTag

  • Paragraf
  • Heading
  • List
  • Hyperlink
  • Image
  • Table
  • Form

Tag Pendukung

  • Bold, Italic, Underline
  • Meter & Progress
  • Br & Hr
  • Pre & Mark
  • Sup & Sub
  • Dialog
  • Code & XMP

Tag di HTML 5

  • Caption & Colgroup

Penggunaan NamaTag

Tag <p>

Digunakan untuk membuat sebuah isi / konten menjadi sebuah paragraf.

Kodenya:

<p>Lorem ipsum dolor, sit amet consectetur adipisicing elit. Vitae, veritatis reiciendis excepturi dolore, deleniti doloribus consequatur temporibus molestiae asperiores soluta sed officia atque rem nihil est sit. Quas perferendis, doloremque.</p>

Outputnya:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Doloribus est dolore neque, architecto, eveniet laboriosam, eligendi aliquam facilis atque maxime illo consectetur perferendis. Ut quisquam provident modi dignissimos, excepturi quasi!

Tag <h1> | <h2> | <h3> | <h4> | <h5> | <h6>

Digunakan untuk menentukan ukuran besar kecilnya sebuah teks. Semakin besar angka pada tag-nya, maka semakin kecil ukuran teksnya.

Kodenya:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Outputnya:

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

Tag <ol> | <ul> | <dl>

Ketiga tag ini digunakan untuk membuat sebuah list. Nah, untuk ketiga tag list tersebut, <ol> dan <ul> mempunyai tag tambahan, yaitu tag <li>. Sedangkan, tag <dl> tambahan tag-nya yaitu <dt> dan <dd>.

<ol> (order list) digunakan untuk membuat list secara terurut. Nilai pada atribut type yaitu: 1, a, A, i dan I.

Kodenya:

<ol>
  <li>Senen</li>
  <li>Selasa</li>
  <li>Rabu</li>
</ol>

Outputnya:

  1. Senen
  2. Selasa
  3. Rabu

<ul> (un-order list) digunakan untuk membuat list tak berurut. Nilai pada atribut type yaitu: disc, square dan circle.

Kodenya:

<ul>
  <li>Ayam Goreng</li>
  <li>Nasi Goreng</li>
  <li>Mie Goreng</li>
</ul>

Outputnya:

  • Ayam Goreng
  • Nasi Goreng
  • Mie Goreng

<dl> (definision list) digunakan untuk mendefinisikan sebuah pada suatu list.

Kodenya:

<dl>
  <dt>Deskripsi Judul</dt>
  <dd>Mendeskripsikan kontennya</dd>
</dl>

Outputnya:

Deskripsi Judul
Mendeskripsikan kontennya

Tag <a>

Digunakan untuk merubah teks menjadi sebuah teks link. Tag <a> membutuhkan tambahan atribut, yaitu href dan juga ada atribut satu lagi yang tidak kalah penting, yaitu target yang fungsinya ketika di klik maka akan membuat tab baru pada browser.

Selengkapnya

Atribut target banyak sekali nilainya, seperti: _blank | _self | _top | _parent

Ada juga atribut download, digunakan sebagai link download.

Kodenya:

<a href="https://www.instagram.com" target="_blank">Klik disini</a>

Outputnya:

Klik disini

Tag <img>

Digunakan untuk menampilkan sebuah gambar atau logo. Tag <img> mempunyai atribut khusus, yaitu src dengan nilai nama gambar tersebut atau URL eksternal. Atribut lainnya seperti: alt | title | width | height

Tag <img> mempunyai ekstension gambar, seperti: png | jpg | jpeg | svg | gif

Selengkapnya

Jika kalian memakai gambar atau logo yang ada di browser, besar kemungkinan akan terkena Copyright. Oleh karena itu, buat gambar atau logo sendiri dengan software atau cari di browser gambar yang bebas pakai atau kunjungi website yang menyediakan gambar atau logo yang bebas pakai.

Kodenya:

<img src="image.png" alt="Tester" title="Tester" width="250" height="200">

Outputnya:

Tag <table>

Digunakan untuk membuat sebuah tabel. Didalam tag <table> terdapat tag tambahan, yaitu: <thead> | <tbody> | <tr> | <th>. Tag <table> mempunyai atribut khusus juga, yaitu: border | cellspacing | cellpading | colspan | rowspan.

Terminologi atau istilah pada tag <table>, yaitu: field (kolom) | record (baris)

Kodenya:

<table border="1" cellspacing="0" cellpadding="5">
  <thead>
    <tr>
      <th></th>
      <th>Kolom 1</th>
      <th>Kolom 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th>Baris 1</th>
      <th>baris 1, kolom 1</th>
      <th>baris 1, kolom 2</th>
    </tr>
    <tr>
      <th>Baris 2</th>
      <th>baris 2, kolom 1</th>
      <th>baris 2, kolom 2</th>
    </tr>
  </tbody>
</table>

Outputnya:

Kolom 1 Kolom 2
Baris 1 baris 1, kolom 1 baris 1, kolom 2
Baris 2 baris 2, kolom 1 baris 2, kolom 2

Rowspan digunakan untuk membuat field baru, sedangkan colspan digunakan untuk membuat record baru.

Kodenya:

<table border="1" cellspacing="0" cellpadding="5">
  <tr>
    <th rowspan="3">Rowspan 3</th>
    <th colspan="2">Cosplan 2</th>
  </tr>
  <tr>
    <th>2,1</th>
    <th>2,2</th>
  </tr>
  <tr>
    <th>3,1</th>
    <th>3,2</th>
  </tr>
</table>

Outputnya:

Rowspan 3 Cosplan 2
2,1 2,2
3,1 3,2

Tag <form>

Digunakan untuk membuat sebuah form. Tag tambahannya seperti: <input> | <label> | <select> | <button> | <textarea> dan masih banyak lagi. Pada tag <select> terdapat tag tambahan lagi, yaitu: <option>.

Tag <button> mempunyai atribut type yang nilainya: submit | reset | button

Selengkapnya
Tag <input> memiliki banyak sekali nilainya pada atribut type, seperti:
*- text, untuk menuliskan sebuah teks
*- email, untuk menuliskan sebuah email
*- password, untuk menuliskan sebuah password
*- radio, hanya memilih satu dari beberapa pilihan
*- checkbox, dapat memilih lebih dari satu pilihan
*- date, dapat menentukan tanggal lahir
*- time, dapat menentukan waktu
*- datetime-local, dapat menentukan tanggal lahir dan waktu
*- number, dapat menentukan angka yang dipilih
*- month, dapat menentukan tahun dan bulan
*- week, dapat menentukan hari kosong atau jadwal kegiatan
*- color, dapat memilih berbagai warna secara acak
*- file, dapat menaruh sebuah file
*- range, membuat bar persentase
*- search, dapat menuliskan sesuatu yang ingin dicari

Atribut action untuk memberikan tujuan akhir dari tag <form> dari sebuah nilai yang dituju, seperti URL atau lainnya.

Atribut method nilainya: post | get

Atribut accept untuk menentukan file apa yang dapat diterima, nilainya: image/* | audio/* | video/*

Atribut autocomplete untuk memperlihatkan histori penulisan yang sebelumnya diketik user. Nilainya: on | off

Atribut autofocus untuk memfokuskan titik kursor mousenya.

Kodenya

<form action="...">
  <table border="1" cellspacing="0" cellpadding="5">
    <tr>
      <th><label for="nama">Nama</label></th>
      <th><input type="text" id="nama"></th>
    </tr>
    <tr>
      <th><label for="email">Email</label></th>
      <th><input type="email" id="email"></th>
    </tr>
    <tr>
      <th><label for="pw">Password</label></th>
      <th><input type="password" id="pw"></th>
    </tr>
    <tr>
      <th><label for="id">Nomor ID</label></th>
      <th><input type="number" min="0" max="5" id="id"></th>
    </tr>
    <tr>
      <th colspan="2">
        <label for="pria">Pria</label><input type="radio" id="pria" name="jeniskelamin">
        <label for="wanita">Wanita</label><input type="radio" id="wanita" name="jeniskelamin">
      </th>
    </tr>
    <tr>
      <th colspan="2">
        <label for="fe">FrontEnd</label><input type="checkbox" id="fe">
        <label for="be">BackEnd</label><input type="checkbox" id="be">
        <label for="fs">Full Stack</label><input type="checkbox" id="fs">
      </th>
    </tr>
    <tr>
      <th><input type="date"></th>
      <th><input type="time"></th>
    </tr>
    <tr>
      <th>Tempat Lahir</th>
      <th>
        <select>
          <option>Jakarta</option>
          <option>Bandung</option>
          <option>Tangerang</option>
        </select>
      </th>
    </tr>
    <tr>
      <th><label for="color">Warna Kesukaan</label></th>
      <th><input type="color" id="color"></th>
    </tr>
    <tr>
      <th><label for="bi">Bahasa Inggris</label></th>
      <th><input type="range" min="0" max="100" id="bi"></th>
    </tr>
    <tr>
      <th colspan="2"><input type="file" accept="image/*"></th>
    </tr>
    <tr>
      <th><label for="komen">Alasan</label></th>
      <th><textarea id="komen"></textarea></th>
    </tr>
    <tr>
      <th><button type="submit">Submit</button></th>
      <th><button type="reset">Reset</button></th>
    </tr>
  </table>
</form>

Tag <form> dengan bantuan tag <table>, agar lebih rapih dan tertata

Outputnya:

Tempat Lahir

Tag Pendukung

Tag <b> | <i> | <u>

Tag <b> digunakan untuk menebalkan teks, tag <i> digunakan untuk memiringkan teks, dan tag <u> digunakan untuk memberi garis bawab pada teks.

Kodenya:

<b>Selamat datang</b> <i>di website</i> <u>BangNomad</u>

Outputnya:

Selamat datang di website BangNomad

Tag <meter> | <progress>

Kedua tag tersebut sama sama digunakan untuk membuat sebuah bar persentasi. Tag ini juga mempunyai atribut khusus, yaitu: min | max | value

Kodenya:

<p>HTML : <meter min="0" max="10" value="7"></meter></p>
<p>CSS : <meter min="0" max="100" value="55"></meter></p>
<p>Downloading : <progress min="0" max="10" value="9"></progress></p>

Outputnya:

HTML :

CSS :

Downloading :

Tag <br> | <hr>

Digunakan sebagai tag pendukung di HTML. Tag <br> untuk membuat garis baru pada sebuah paragraf, sedangkan tag <hr> untuk membuat sebuah garis pembatas secara horizontal.

Kodenya:

<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. <br><br>Ratione perferendis impedit at modi dicta repellendus nulla cupiditate commodi debitis ducimus!<hr> Lorem ipsum dolor sit amet consectetur adipisicing elit. Deserunt, illum reiciendis!</p>

Outputnya:

Lorem ipsum dolor sit amet consectetur adipisicing elit.

Ratione perferendis impedit at modi dicta repellendus nulla cupiditate commodi debitis ducimus!

Lorem ipsum dolor sit amet consectetur adipisicing elit. Deserunt, illum reiciendis!

Tag <pre> | <mark>

Tag <pre> digunakan untuk custom teks lebih bebas dibanding tag <p>, misal memberi spasi lebih panjang. Sedangkan, tag <mark> digunakan untuk membuat label bewarna kuning (defaultnya) pada sebuah teks.

Kodenya:

<pre>Selamat Datang      di Website     <mark>BangNomad</mark></pre>

Outputnya:

Selamat Datang      di Website     BangNomad

Tag <sup> | <sub>

Digunakan untuk membuat teks menjadi berpangkat. Dan tag ini cocok untuk kalkulasi perhitungan. Tag <sup> untuk pangkat keatas, sedangkan tag <sub> untuk pangkat kebawah.

Kodenya:

<p>2<sup>2</sup> + 2<sup>3</sup> = 12 <br> H<sub>2</sub>O adalah ...</p>

Outputnya:

22 + 23 = 12
H2O adalah ...

Tag <dialog>

Digunakan untuk membuat sebuah kotak dialog antar user dengan program. Tag <dialog> memiliki atribut tambahan, yaitu: open

Kodenya:

<dialog open>Selamat Datang!</dialog>

Outputnya:

Selamat Datang!

Tag <code> | <xmp>

Digunakan untuk menampilkan sebuah source code yang kita koding ke dalam halaman website.

Kodenya:

<code>console.log('ok');</code>
<xmp><p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Architecto, beatae.</p></xmp>

Output Tag <code>:

console.log('ok!');

Output tag <xmp>:

<p>Lorem, ipsum dolor sit amet consectetur adipisicing elit. Sed, provident.</p>

HTML 5

Tag <caption> dan <colgroup>

Tag <caption> digunakan untuk memberi judul pada sebuah tabel. Sedangkan tag <colgroup> digunakan untuk memberi style pada sebuah tabel, dan tag <colgroup> memiliki tag tambahan, yaitu: <col> dan didalamnya memiliki atribut span

<table border="1">
  <caption>Tabel Tester</caption>
  <colgroup>
    <col span="1" style="background-color: #0ff;">
    <col span="2" style="background-color: #f0f;">
    <col span="1" style="background-color: #ff0;">
  </colgroup>
  <tr>
    <th>ID</th>
    <th>Nama</th>
    <th>Umur</th>
    <th>Tempat</th>
  </tr>
  <tr>
    <th>1</th>
    <th>Rahmat</th>
    <th>22</th>
    <th>Bandung</th>
  </tr>
  <tr>
    <th>2</th>
    <th>Ismet</th>
    <th>19</th>
    <th>Tangerang</th>
  </tr>
  <tr>
    <th>3</th>
    <th>Ci'u Wu</th>
    <th>28</th>
    <th>Bogor</th>
  </tr>
</table>

Outputnya:

Tabel Tester
ID Nama Umur Tempat
1 Dadang 22 Bandung
2 Ismet 19 Tangerang
3 Ci'u Wu 28 Bogor

Tag <details>

Digunakan untuk menyembunyikan konten dan akan muncul ketika di klik data lengkapnya. Tag <details> memiliki tag tambahan, yaitu: <summary>

Kodenya:

<details>
  <summary>Details</summary>
  <p>Lorem ipsum, dolor sit amet consectetur adipisicing elit. Reiciendis, laudantium! Atque, blanditiis, at expedita repudiandae inventore cumque suscipit provident rerum mollitia sint iusto nisi sapiente.</p>
</details>

Outputnya:

Details

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Reiciendis, laudantium! Atque, blanditiis, at expedita repudiandae inventore cumque suscipit provident rerum mollitia sint iusto nisi sapiente.

Tag <datalist>

Digunakan untuk membantu tag <input> dengan cara menuliskannya. Hal ini sama seperti tag <select> dan sama memiliki tag tambahan, yaitu: <option>

Kodenya:

<input list="kota">
<datalist id="kota">
  <option value="Jakarta"></option>
  <option value="Tangerang"></option>
  <option value="Bandung"></option>
</datalist>
<button type="submit">Kirim</button>

Outputnya:

Tag <figure>

Digunakan untuk membantu tag <img> agar mempermudah saat kolaborasi dengan bahasa lainnya, seperti bahasa CSS. Tag <figure> memiliki tag tambahan, yaitu: <figcaption>

Kodenya:

<figure>
  <img src="image.png" alt="Background">
  <figcaption>Background</figcaption>
</figure>

Outputnya:

Tag <fieldset>

Digunakan untuk untuk membantu tag <form> dan mempercantik sedikit style-nya tanpa CSS. Tag <fieldset> memiliki tag tambahan, yaitu: <legend>

Kodenya:

<form action="...">
  <fieldset>
    <legend>Personal Info</legend>
    First Name : <input type="text"> <br>
    Last Name : <input type="text">
  </fieldset>
</form>

Outputnya:

Personal Info First Name :
Last Name :

Tag <iframe>

Digunakan untuk menampilkan sebuah file, teks, atau gambar ke dalam halaman website dan membutuhkan file lokal terpisah jikalau menampilkannya secara lokal. Jika ingin menampilkan konten secara eksternal, maka salin URL eksternalnya.

Kodenya:

<iframe src="index.html" frameborder="1"></iframe>

Outputnya:

Tag <iframe> jika bersama tag <a>, maka akan muncul kontennya saat di klik.

Kodenya:

<a href="index.html" target="myFrame">iframe</a> <br>
<iframe name="myFrame" frameborder="1"></iframe>

Outputnya:

iframe

Tag <audio>

Digunakan untuk memutar musik didalam website atau untuk mendengarkan lagu lagu yang diputar. Tag <audio> memiliki tag tambahan, yaitu <source>

Tag <audio> memiliki beberapa ekstension file, seperti: .mp3 | .ogg | .wav | .mpeg

Selengkapnya

Tag <audio> sama seperti tag <video> yang memiliki atribut seperti:
Atribut autoplay, digunakan untuk memulai video saat pertama kali membuka website
Atribut controls, digunakan untuk menampilkan kontrol
Atribut loop, digunakan untuk mengulang ulang video tanpa henti
Atribut muted, digunakan untuk memute suara dari video tersebut

Kodenya:

<audio controls loop>
  <source src="music.mpeg" type="audio/mpeg">
</audio>

Outputnya:

Tag <video> | <embed>

Digunakan untuk menampilkan sebuah video. Kedua tag tersebut memiliki atribut tambahan, seperti: autoplay | controls | loop | muted | preload | src | poster | width | height

Tag <video> memiliki beberapa ekstension, seperti: .mp4 | .mkv dan lain lain.

Selengkapnya

Atribut autoplay, digunakan untuk memulai video saat pertama kali membuka website
Atribut controls, digunakan untuk menampilkan kontrol
Atribut loop, digunakan untuk mengulang ulang video tanpa henti
Atribut muted, digunakan untuk memute suara dari video tersebut
Atribut poster, digunakan untuk merubah thumbnail video menjadi objek yang ditentukan

Kodenya:

<video controls loop poster="...">
  <source src="video.mp4" type="video/mp4">
</video>

Outputnya:
