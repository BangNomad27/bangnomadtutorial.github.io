Untuk dasarnya saya akan memperkenalkan nama nama Tag didalam HTML. Didalam HTML terdapat 3 urutan pada syntax-nya pertama untuk namaTag-nya, kedua nama atribut dan ketiga ialah nilai dari atributnya.

Ambil Template

Tag yang sering digunakan
*- <div> dan <span>
Kedua tag itu tidak memiliki arti apa apa hanya digunakan untuk mengelompokkan tag HTML di dalamnya, dan tag tersebut disebut parent-nya.
Dan <div> ber-display block, sedangkan <span> ber-display inline. Pertanyaannya, apa itu display? Display adalah nama property di CSS, dan untuk lebih lanjut mengenai display itu apa, cek di Tutorial CSS.
Tag dasar yang harus diketahui
*- <html>
Tag pembuka saat melakukan coding
*- <title>
Tag untuk me-rename judul titel halaman web
*- <style>
Tag untuk menuliskan CSS secara internal
*- <meta>
Tag metadata dokumen pada HTML
*- <head>
Tag bagian atas (kepala) dari coding
*- <body>
Tag bagian badannya untuk menuliskan codenya
*- <script>
Tag untuk menuliskan script Javascript secara internal

Atribut pada HTML

Atribut yang sering digunakan
*- id
Untuk menentukan id unik pada sebuah elemen (namaTag)
*- class
Untuk memberi kelas pada sebuah elemen. Atribut ini akan berguna ketika banyak bermain CSS
Beberapa nama atribut lainnya
*- dir
Menentukan arah teksnya. Nilainya: ltr (left to right) atau rtl (right to left)
*- lang
Menentukan bahasa yang ingin dipakai pada sebuah website nantinya
*- style
Memberikan style CSS secara inline
*- translate
Membuat konten elemen dapat diterjemahkan
*- title
Memberi informasi tambahan pada elemen tersebut
*- alt
Mendefinisikan sebuah teks pada gambar yang hilang atau tidak tampil
*- hidden
Untuk menyembunyikan elemen tersebut
*- accesskey
Menentukan tombol shortcut aktif atau fokus pada elemennya aja
*- href
Menentukan arah (tujuan) ketika di klik
*- target
Menentukan targetnya (arah) saat di klik. Biasanya sinkron dengan atribut name
*- src
Untuk menampilkan suatu objek melalui URL Eksternal maupun Internal
*- action
Memberikan aksi ketika tombol submit di klik
*- for
Memberi tujuan tag <label> ketika teks di klik. Biasanya sinkron dengan atribut id di tag <input>
*- name
Menentukan nama ketika menggunakan tag yang berbeda tapi nilainya sama
*- placeholder
Memberikan sebuah teks didalam tag <input> dan <textarea>
*- width - height
Memberikan ukuran panjang x lebar pada sebuah gambar di tag <img>
*- controls
Memberikan kontrol ketika menggunakan tag <audio> dan <video>
*- autoplay
Untuk menjalankan suara atau video ketika pertama kali buka website

Semantic Elements

Semantic element terbagi menjadi 2 bagian, yaitu non-semantic dan semantic.

Non-semantic ialah mendefinisikan sesuatu tentang isinya, contohnya seperti <div> dan <span>. Sedangkan semantic ialah jelas mendefinisikan isinya, contohnya seperti <form>, <img> dan <table>.

Nah di HTML 5 banyak namaTag baru dan membuang namaTag yang sudah tak terpakai lagi oleh developernya, tapi saya akan memberi tau namaTag barunya aja.

Mengapa semantic element di HTML 5? Karena namaTag pada HTML 5 dibuat dengan unsur untuk mempermudah dalam membuat tagnya, serta jauh lebih jelas dan efisien kodingan.

Menurut W3C "Memungkinkan data yang akan dibagi dan di gunakan kembali diseluruh aplikasi, perusahaan, dan masyarakat".

Tag baru di HTML 5
*- <article>
Mendefinisikan sebuah artikel
*- <aside>
Mendefinisikan sebuah konten selain dari konten utama
*- <bdi>
Mendefinisikan sebuah bagian dari teks yang mungkin di format dalam arah yang berbeda dari teks lainnya
*- <details>
Mendefinisikan rincian tambahan bahwa pengguna dapat melihat atau menyembunyikan konten
*- <dialog>
Mendefinisikan sebuah kotak dialog atau jendela dialog
*- <figcaption>
Mendefinisikan sebuah caption untuk tag <figure>
*- <figure>
Menentukan konten mandiri, seperti ilustrasi, diagram, foto, daftar kode, dan lain lain
*- <footer>
Mendefinisikan sebuah footer untuk dokumen atau bagian bawah website
*- <header>
Menentukan header untuk dokumen atau bagian tertentu
*- <main>
Menentukan isi/konten utama pada sebuah dokumen
*- <mark>
Menandai teks konten yang di anggap penting
*- <menuitem>
Mendefinisikan item perintah atau menu bahwa pengguna dapat memanggil dari menu popup
*- <meter>
Mendefinisikan pengukuran skalar dalam kisaran yang dikenal (agauge)
*- <nav>
Mendefisinikan list navigasi atau disebut juga navbar
*- <progress>
Mendefinisikan seperti bar loadings atau digunakan juga untuk loading pen-download-an
*- <ruby>
Mendefinisikan sebuah ruby penjelasan (untuk tipografi Asia Timur)
*- <rp>
Mendefinisikan apa yang harus ditampilkan di browser yang tidak mendukung <ruby> penjelasan
*- <rt>
Mendefinisikan sebuah penjelasan atau pengucapan karakter (untuk tipografi Asia Timur)
*- <section>
Mendefinisikan sebuah bagian dalam sebuah dokumen
*- <summary>
Mendefinisikan sebuah judul terlihat untuk tag <details>
*- <time>
Mendefinisikan tanggal dan waktu
*- <wbr>
Mendefinisikan sebuah kemungkinan garis-break
Tag baru di HTML untuk <form>
*- <datalist>
Mendefinisikan pilihan yang telah ditentukan untuk kontrol input
*- <keygen>
Mendefinisikan bidang kunci-pasangan generator (for forms)
*- <output>
Mendefinisikan hasil perhitungan
Tag baru di HTML 5 untuk Grafik
*- <canvas>
Mendefinisikan gambar grafis menggunakan javascript
*- <svg>
Mendefinisikan gambar grafis menggunakan SVG
Tag baru di HTML 5 untuk media
*- <audio>
Mendefinisikan sebuah suara atau musik
*- <embed>
Mendefinisikan wadah untuk aplikasi eksternal (like plug-ins)
*- <source>
Mendefinisikan sumber untuk <video> dan <audio>
*- <track>
Mendefinisikan lagu untuk <video> dan <audio>
*- <video>
Mendefinisikan video atau film konten
