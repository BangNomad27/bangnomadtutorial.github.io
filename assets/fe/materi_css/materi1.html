Mulai

Apa itu selector? Selector adalah namaTag di HTML yang dipanggil namaTagnya dan disebut sebagai selector. Selector bisa juga memanggil nama atribut class atau id didalam namaTag tersebut dan memanggilnya menggunakan simbol selector atau disebut juga pseudo-class.

Property CSS 1

Font Size

Digunakan untuk menentukan ukuran huruf pada sebuah teks.

Kodenya:

selector { font-size: 22px; }

Outputnya:

Lorem ipsum dolor sit amet consectetur adipisicing facilis!

Font Family

Digunakan untuk menentukan jenis hurufnya yang ditentukan.

Kodenya:

selector { font-family: georgia, sans-serif; }

Outputnya:

Lorem ipsum dolor sit, amet consectetur adipisicing, elit. Nemo, veniam!

Font Weight

Digunakan untuk menentukan ketebalan hurufnya, bedanya dengan tag <b> ialah property ini bisa mengatur ketebalan hurufnya. Nilainya: normal | bold | bolder | 500-900 dan lain lain.

Kodenya:

selector { font-weight: 900; }

Outputnya:

Lorem ipsum, dolor sit amet, consectetur adipisicing elit.

Font Style

Digunakan untuk menentukan kemiringan pada huruf, dan property ini sama seperti tag <i> pada HTML. Nilainya: normal | italic | oblique dan lain lain.

Kodenya:

selector { font-style: oblique; }

Outputnya:

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Consectetur, ipsum.

Font Variant

Digunakan untuk membuat huruf menjadi capital walaupun teks default ga capital. Nilainya: normal | small-caps | inherit | unset dan lain lain.

Kodenya:

selector { font-variant: small-caps; }

Outputnya:

Lorem ipsum dolor sit amet consectetur, adipisicing elit voluptates!

Masih banyak lagi property font lainnya, seperti: font-kerning | font-synthesis | font-stretch dan lain lain.

Text Align

Digunakan untuk menentukan kerataan pada sebuah teks. Nilainya: left | center | right | justify

Kodenya:

selector { text-align: left; }
selector { text-align: center; }
selector { text-align: right; }

Outputnya:

Rata Kiri

Rata Tengah

Rata Kanan

Text Indent

Digunakan untuk memberi ruang kosong diawal paragraf seperti menggunakan HTML entitas yang ini &nbsp;

Kodenya:

selector { text-indent: 20px; }

Outputnya:

Lorem, ipsum dolor, sit amet consectetur adipisicing elit. Voluptatibus est impedit aliquam eos distinctio maiores quos unde officiis, corporis sit quod minus atque! Non tempora deleniti praesentium officiis perferendis?

Text Decoration

Digunakan untuk mendekorasi sebuah teks dengan menambahkan sebuah garis pada teksnya. Nilainya: none | underline | overline | line-through

Kodenya:

selector { text-decoration: underline; }
selector { text-decoration: overline; }
selector { text-decoration: line-through; }

Outputnya:

Underline | Overline | Line-Through

Text Transform

Digunakan untuk membuat style pada hurufnya. Nilainya: none | lowercase | uppercase | capitalize

Kodenya:

selector { text-transform: lowercase; }
selector { text-transform: uppercase; }
selector { text-transform: capitalize; }

Outputnya:

LOWERCASE

uppercase

capitalize

Masih banyak lagi property text lainnya, seperti: text-anchor | text-shadow | text-orientation | text-rendering | text-overflow dan lain lain.
