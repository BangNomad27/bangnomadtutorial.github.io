Background

Kedua property ini digunakan untuk memberikan warna terhadap elemennya. Background digunakan untuk memberi warna terhadap latarnya.

Selengkapnya

background-color

Untuk memberikan warna terhadap halaman website atau background websitenya. Nilainya: nama_warna | hexa-desimal | rgb() dan lain lain.

Kodenya:

selector { background-color: grey; }

background-image

Untuk memberikan sebuah gambar pada background. Nilainya: url() dan lain lain.

Kodenya:

selector { background-image: url(../img/image.png); }

background-size

Untuk menentukan ukuran sebuah gambarnya. Nilainya: count manually | cover dan lain lain.

Selengkapnya

Count Manually ialah sebuah nilai yang diatur manual oleh programmernya. Ini terdapat dua nilai, yaitu nilai pertama untuk menentukan nilainya secara horizontal, sedangkan nilai kedua untuk menentukan nilainya secara vertikal. Seperti contoh dibawah ini:

Kodenya:

selector { background-size: 200px 250px; }

background-repeat

Untuk menentukan gambarnya berulang atau single image. Nilainya: no-repeat

Kodenya:

selector { background-repeat: no-repeat; }

background-position

Untuk menentukan posisi backgroundnya. Nilainya: count manually | left | right | center dan lain lain.

Selengkapnya

Count Manually ialah sebuah nilai yang diatur manual oleh programmernya. Ini terdapat dua nilai, yaitu nilai pertama untuk menentukan nilainya secara horizontal, sedangkan nilai kedua untuk menentukan nilainya secara vertikal.

Kodenya:

selector { background-position: 0 -100px; }

Dan masih banyak lagi property untuk background ini, seperti: background-clip | background-attachment | background-blend-mode | background-origin dan lain lain.

Pseudo-Class

Pseudo-class bukanlah sebuah property, namun ini adalah simbol selector. Pseudo-class sangatlah penting pada bahasa CSS, berikut beberapa simbol selector yang bisa kami jelaskan:

Selengkapnya

*

Simbol selector bintang (*) digunakan untuk mengubah keseluruhan property pada CSS. Namun hati hati jika menggunakan simbol selector ini, karena bisa dibilang salah satu untuk meng-reset CSS dan biasanya juga programmer me-reset margin dan padding aja.

Kodenya:

* { color: blueviolet; }

#x

Simbol selector pagar (#) digunakan untuk memberi style pada namaTag yang memiliki atribut id. Namun perlu hati hati juga, karena simbol pagar memiliki nilai spesificity yang sangat besar.

Kodenya:

#container { color: rgb(17,17,17); }

.x

Simbol selector titik(.) digunakan untuk memberi style pada namaTag yang memiliki atrbut class.

Kodenya:

.container { font-size: 27px; }

x y

Simbol selector spasi digunakan untuk menargetkan ke tujuan yang lebih spesifik.

Kodenya:

#container .content h1 { font-weight: normal; }

x, y

Simbol selector koma (,) digunakan untuk menargetkan beberapa selector yang ingin diubah lebih dari satu.

Kodenya:

aside, article { width: 600px; }

x:hover

Simbol selector :hover digunakan untuk mengubah style pada saat kursor mengarah ke elemen yang diberikan :hover tersebut.

Kodenya:

a:hover { color: coral; }

Outputnya:

Hover Me!

x:active

Simbol selector :active digunakan untuk mengubah style pada saat menekan elemen sembari ditahan kursornya, maka elemen yang diberi :active akan berubah.

Kodenya:

x:visited

Simbol selector :visited digunakan untuk mengubah style pada saat si user telah mengunjungi atau meng-klik elemen tersebut.

Kodenya:

Simbol selector tambah (+) digunakan untuk mengubah style dengan memilih elemen terdekat setelahnya dan mengabaikan elemen sebelumnya.

Property CSS 2

Display

Display adalah sebuah display yang ada di setiap namaTag, seperti <div> secara default memiliki display bernilai block. Ada beberapa nilai pada property display, seperti: none | inline | block | inline-block dan lain lain.

Selengkapnya
Penjelasan mengenai nilai pada display:
*- none, untuk menghilangkan sebuah elemen
*- inline, tidak dapat menambahkan garis baru ketika dibuat (secara default)
*- block, dapat mengatur ukurang pada elemennya
*- inline-block, dapat menambahkan garis baru serta mengatur ukurannya

Kodenya:

selector { display: inline-block; }

Overflow

Overflow adalah sebuah property yang digunakan untuk mengatur perilaku elemen yang kasusnya keluar dari parent-nya. Nilainya, yaitu: visible | auto | hidden | scroll dan lain lain.

Apa itu parent? Parent adalah sebuah pembungkus atau disebut box model yang didalamnya memiliki beberapa namaTag dan namaTag itu disebut juga sebagai anak.

Selengkapnya
Penjelesan nilai pada property overflow:
*- visible, adalah nilai default disetiap property
*- auto, CSS akan otomatis menambahkan scroll ketika kontennnya ga cukup, tapi jika cukup maka scroll akan hilang
*- hidden, CSS akan menyembunyikan konten ketika melebihi parentnya
*- scroll, memunculkan sebuah scroll secara seperti nilai auto, bedanya ketika kontennya cukup scrollnya tetep ada

Kodenya:

selector { overflow: auto; }

Float

Float adalah property CSS yang bertugas mengatur posisi sebuah elemen dan float memiliki tiga nilai: none | left | right. Lalu ada dua hal pada float, yaitu: normal-flow dan out-of-flow.

Selengkapnya
Floar dapa melakukan:
*- Text-Wrapping, membuat teks mengelilingi gambar atau elemen
*- Image Gallery, membuat serangkaian gambar menjadi sebuah galeri
*- Multi-Column Layout, membuat halaman memiliki beberapa kolom, dan lain lain

Kodenya:

selector { float: left; }

Outputnya:

Tapi kekurangan dari property float ialah konten atau anak dari parent-nya akan collapse atau keluar dari parent-nya, yang membuat box model tersebut berantakan. Oleh karena itu, property float dibantu dengan property clear.

Clear

Property clear bertugas untuk membersihkan atau menghentikan property float. Karena jika tidak dihentikan, maka akan berantakan dan tidak responsive, seperti yang kalian lihat output diatas. Nilainya: left | right | both

Selengkapnya
Property clear terdapat tiga nilai, yaitu:
*- left, digunakan ketika selector mempunyai float bernilai 'left'
*- right, digunakan ketika selector mempunyai float bernilai 'right'
*- both, digunakan ketika selector mempunyai float bernilai 'left' dan 'right'
Terdapat tiga cara untuk membersihkan atau menghentikan property float, yaitu:
*- Menggunakan property overflow yang diberi nilai'auto'
*- menggunakan <div> kosong
*- menggunakan teknik micro clearfix

Untuk ngakalin dengan micro clearfix, ambil syntax dibawah ini:

Kodenya:

/* Micro Clearfix */
.cf:before,
.cf:after {
 content: " ";
 display: table;
}
.cf:after {
 clear: both;
}
.cf {
 *zoom: 1;
}

Setelah copy dan paste syntax diatas, kalian pergi ke HTML dan cari namaTag yang memiliki property float didalamnya dan tambah kelas .cf

Contoh namaTag <div>:

<div class="container cf">

Kodenya:

selector { clear: left; }

Position

Position adalah property penting juga tapi agak sulit untuk dipelajari oleh pemula. Position dapat melakukan tata letak dari elemen atau box model (parent-nya), sesuai dengan keinginan sang programmernya. Nilainya: static | relative | absolute | fixed dan lain lain.

Berikut penjelasan nilai pada property position:

Static (nilai default):

Selengkapnya
*- Jika tidak memberikan property position pada elemen, maka static adalah nilai defaultnya
*- Jika menggunakan selain position static, disebut sebagai non-static. Maka, akan membuat elemen menjadi berbeda dimensi dari elemen lainnya
*- Elemen yang diberikan position non-static, dapat menggunakan property tambahan, yaitu: top | right | bottom | left

Kodenya:

selector { position: static; }

Outputnya:

Relative

Selengkapnya
*- Ketika menggerakkan elemen dengan posisi relative, maka ruang yang ditempati oleh elemen tersebut tetep masih ada
*- Ketika menggerakkan elemen dengan posisi relative, maka elemen akan bergerak relatif terhadap posisi semulanya
*- Jika memberikan property top dan left dengan nilai nol/null, maka elemen tersebut ga berubah dari posisi awalnya

Kodenya:

selector {
  position: relative;
  top: 2rem; left: 2rem;
}

Outputnya:

Absolute

Selengkapnya
*- Ketika menggerakkan elemen dengan posisi absolute, maka ruang yang ditempati oeh elemen tersebut dianggap tidak ada
*- Ketika menggerakkan elemen dengan posisi absolute, maka elemennya bergerak relatif terhadap posisi parent-nya/box modelnya
*- Jika memberika property top dan left dengan nilai nol/null, maka elemen akan menempel pada parent-nya (default parent ialah <body>)

Kodenya:

selector {
  position: absolute;
  top: 10px; left: 10px;
}

Outputnya:

Fixed

Selengkapnya
*- Ketika menggerakkan elemen tersebut menggunakan top, right, bottom, left, maka dia akan selalu bergerak relatif terhadap window-nya si browser (window si browser adalah tag <body>)
*- Ketika di scroll halamannya, maka elemen tersebut akan terkunci diposisi tertentu

Kodenya:

selector { position: fixed; }

Outputnya:

Outputnya biasanya seperti navbar yang fix diatas atau bawah browser, dan selalu ikut ketika di scroll halamannya. Contoh lainnya seperti tombol dipojok kanan bawah pada website saya ini, karena agar mempermudah tanpa perlu scroll keatas lagi.

Nilainya sebuah angka tanpa satuan, semakin besar angka nilainya semakin menonjol atau paling depan elemen tersebut.

Kodenya:
